revolia

Anonymiseur de documents 100 % local · hors-ligne · rien n'est envoyé

Déposez un document (PDF, Word, image — même scanné), l'outil repère les informations identifiantes, vous validez, puis vous exportez une version anonymisée à utiliser sans risque dans une IA. Tout le traitement se fait dans votre navigateur — aucune donnée ne quitte votre appareil.
